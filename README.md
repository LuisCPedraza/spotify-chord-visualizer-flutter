# Spotify Chord Visualizer (Flutter)

Aplicacion movil en Flutter que integra Spotify y presenta acordes sincronizados con una interfaz minimalista orientada a practica musical.

## Highlights

- Integracion con Spotify (arquitectura y plan de autenticacion API).
- Visualizacion de acordes sincronizados (boceto funcional navegable).
- Diseno minimalista inspirado en principios de claridad visual.
- Documentacion completa de producto, Scrum, arquitectura, pruebas y roadmap.

## Estructura del repositorio

- [app_movil_spotify](app_movil_spotify): proyecto Flutter principal.
- [app_movil_spotify/docs](app_movil_spotify/docs): documentacion del proyecto.

## Demo de boceto interactivo

El mockup navegable del flujo visual esta en:

- [app_movil_spotify/docs/12_BOCETO_VISUAL_APP.dart](app_movil_spotify/docs/12_BOCETO_VISUAL_APP.dart)

Para ejecutarlo:

```bash
cd app_movil_spotify
flutter pub get
flutter run -t docs/12_BOCETO_VISUAL_APP.dart
```
