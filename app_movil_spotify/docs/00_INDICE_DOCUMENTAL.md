# Indice Documental del Proyecto

Este indice centraliza toda la documentacion del proyecto AppMovilSpotify para facilitar revisiones academicas, desarrollo y exposicion final.

## Estructura de documentos

1. [01_PRODUCTO_Y_ALCANCE.md](01_PRODUCTO_Y_ALCANCE.md)
   Documento de vision, problema, objetivos, alcance, requisitos, KPI y criterios de exito.

2. [02_SCRUM_PLAN.md](02_SCRUM_PLAN.md)
   Plan Scrum completo: epicas, historias de usuario, subtareas, estimaciones, sprints y Definition of Done.

3. [03_ARQUITECTURA_TECNICA.md](03_ARQUITECTURA_TECNICA.md)

   Estrategia de pruebas unitarias, widget, integracion, calidad, performance y aceptacion.

7. [07_RIESGOS_CUMPLIMIENTO.md](07_RIESGOS_CUMPLIMIENTO.md)
   Matriz de riesgos, mitigaciones y marco de cumplimiento tecnico/legal.

8. [08_PLAN_ENTREGA_ROADMAP.md](08_PLAN_ENTREGA_ROADMAP.md)

9. [09_RUNBOOK_OPERACION.md](09_RUNBOOK_OPERACION.md)
   Guia operativa para levantar el entorno, configurar credenciales y ejecutar el proyecto.

10. [10_PLANTILLA_ENTREGA_ACADEMICA.md](10_PLANTILLA_ENTREGA_ACADEMICA.md)
   Plantilla formal para entrega final con portada, estructura academica y anexos de evidencia.

11. [11_GUIA_STYLES_MINIMALISTA.md](11_GUIA_STYLES_MINIMALISTA.md)
   Guia visual de estilos minimalistas tipo Apple para aplicar en Flutter.

12. [12_BOCETO_VISUAL_APP.md](12_BOCETO_VISUAL_APP.md)
   Boceto de pantallas y composicion visual del MVP con el estilo seleccionado.

13. [13_EVIDENCIA_SPRINT4_CALIDAD.md](13_EVIDENCIA_SPRINT4_CALIDAD.md)
   Evidencia de calidad del cierre: pruebas, script de demo y cierre de riesgos.
